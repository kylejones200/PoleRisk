# Examples

This directory contains example scripts, benchmarks, and demonstrations.

## Benchmarks

- **benchmark_numba_acceleration.py** - Benchmark Numba JIT performance improvements
- **benchmark_rust_vs_python.py** - Compare Rust extension performance vs pure Python

## Demonstrations

- **demo_ml.py** - Machine learning pipeline demonstration
- **visualize_pole_health.py** - Pole health visualization examples

## Usage

Run any example from the project root:

```bash
python examples/demo_ml.py
python examples/benchmark_numba_acceleration.py
```

These scripts are for testing, demonstration, and performance analysis purposes.
